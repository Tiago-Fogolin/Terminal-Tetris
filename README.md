# Terminal Tetris
Esse projeto é um tetris de terminal feito em python, sem dependências externas.

A ideia era fazer um jogo simples de forma procedural, sem grandes complicações.

Depois vou tentar expandir e criar outras branches, que vão ter outros tipos de organização de código, para tentar ver quais são as vantagens e desvantagens de certos tipos de organização, e se devemos realmente perder tanto tempo com isso.

Ainda faltam algumas adições, que farei com o tempo.
